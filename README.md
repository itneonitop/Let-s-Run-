# Let's Run! - Инструкция по локальному запуску

Эта игра поддерживает мультиплеер через локальную сеть или Hamachi.

## Как запустить у себя (Хост):
1. Убедитесь, что установлен [Node.js](https://nodejs.org/).
2. Распакуйте архив с игрой.
3. Откройте терминал в папке проекта и выполните:
   ```bash
   npm install
   npm run dev
   ```
4. Игра будет доступна по адресу `http://localhost:3000`.

## Как играть с друзьями через Hamachi:
1. Запустите Hamachi и создайте сеть.
2. Друзья должны войти в вашу сеть.
3. Дайте друзьям ваш **IPv4 адрес** из Hamachi.
4. Друзья заходят в браузере по адресу: `http://ВАШ_IP_ИЗ_ХАМАЧИ:3000`.

## Режимы игры:
- **FFA (Free For All)**: Каждый сам за себя.
- **Teams**: Командный забег. Улучшения на развилках работают против чужих команд.
